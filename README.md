# Nova

A minimal iOS browser built with Swift and SwiftUI. Made to show more of your content and less controls.

https://github.com/user-attachments/assets/9b3e1a7c-342b-4444-9e4a-ad58872fc98c

## Features

- **Content First Design** 
- **Minimal TabBar** 
- **Built Natively with SwiftUI** 

## Tech Stack

| Layer | Technology |
|-------|------------|
| UI | SwiftUI |
| Rendering | WebKit |
| Platform | iOS |
| Language | Swift |

## Getting Started

1. Clone the repo
   ```bash
   git clone https://github.com/TopMyster/Nova.git
   ```
2. Open `Nova/Nova.xcodeproj` in Xcode
3. Select an iOS simulator or device
4. Build and run
